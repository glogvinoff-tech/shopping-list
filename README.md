# Список покупок - Express.js приложение

Простое приложение для управления списком покупок, созданное с использованием Express.js.

## Функционал
- Просмотр списка покупок
- Добавление новых товаров
- Изменение количества товаров
- Отметка товаров как купленных/не купленных
- Удаление товаров
- Фильтрация по статусу
